# Data Intelligence - building production-grade intelligence platforms

## Overview

This track focuses on building a modern Data Intelligence platform end-to-end using a lakehouse architecture on the cloud. You will design scalable data pipelines that ingest real-world operational data, transform it into analytics-ready datasets, and deliver governed insights through dashboards and APIs.

Rather than focusing on isolated technical components, this track approaches data engineering as the foundation of data intelligence: enabling reliable, decision-ready information for complex operational environments such as aviation, logistics, and supply chains.

By the end of the track, you will have built a production-style data intelligence system that connects raw data to actionable insight.

## What You Will Build

- A cloud-based lakehouse architecture
- Automated ingestion pipelines for operational data
- Scalable transformation workflows
- Analytics-ready data models
- Data products exposed via APIs
- Interactive dashboards delivering operational intelligence
  ### Optional:
- CI/CD-enabled, production-oriented data workflows
